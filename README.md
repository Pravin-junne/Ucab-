# Ucab - Taxi Booking Application

This project is a full-stack taxi booking application built with:

- Backend: Node.js, Express, MongoDB, Mongoose, JWT, Bcrypt
- Frontend: React (Vite), React Router, Axios, Bootstrap

## Setup

### Backend

1. Copy `.env.example` to `.env` in the `backend` folder and update values.
2. Install dependencies (already done if you used the provided setup):

   ```bash
   cd backend
   npm install
   npm run dev
   ```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The frontend assumes the backend API is running on `http://localhost:5000`.
